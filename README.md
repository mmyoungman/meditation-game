# Meditation Game

Created by Mark Youngman

To build/run you need:
- gcc/g++
- SDL2
- SDL2-image

### For linux
```
$ mkdir build
$ cd build
$ ../code/build_linux.sh
```

### For wasm
Additionally need emscripten installed for `emcc` and `emrun`.

```
$ mkdir build
$ cd build
$ ../code/build_wasm.sh
```
